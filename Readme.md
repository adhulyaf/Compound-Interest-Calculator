##  Features
Takes input for:
  - Principal Amount
  - Annual Interest Rate
  - Number of times interest is compounded per year
  - Number of years
Applies the standard compound interest formula
Displays the final amount with two-decimal precision
Clean and simple console-based output

## Formula Used
Amount = Principal [ 1 + Rate \ TimeCompounded ] TimeCompounded * years 

Where:
  **P** = Principal amount
  **r** = Annual interest rate (decimal form)
  **n** = Times interest is compounded per year
  **t** = Number of years
  **A** = Final amount
